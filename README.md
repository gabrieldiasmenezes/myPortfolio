Bem-vindo ao repositório do meu portfólio profissional.
Este projeto foi desenvolvido para apresentar minhas habilidades, projetos e trajetória focada em **Análise de Dados**, com objetivo de conquistar minha primeira oportunidade de estágio na área.

## 🌐 Sobre o Projeto

Este portfólio é uma aplicação web moderna construída com **Next.js**, com foco em performance, acessibilidade e experiência visual.

Ele apresenta:

* 📊 Projetos de análise de dados
* 🎓 Formação acadêmica
* 🧠 Skills técnicas e comportamentais
* 📬 Formulário funcional de contato
* 🌙 Tema claro/escuro
* 🌍 Suporte bilíngue (PT/EN)

O design foi pensado para transmitir uma identidade **tecnológica, analítica e profissional**, alinhada ao mercado de dados.

## ⚙️ Como Rodar o Projeto

Clone o repositório:

```bash
git clone https://github.com/seuusuario/seurepo.git
```

Instale as dependências:

```bash
npm install
```

Crie um arquivo `.env.local` com:

```
NEXT_PUBLIC_EMAILJS_SERVICE_ID=your_service_id
NEXT_PUBLIC_EMAILJS_TEMPLATE_ID=your_template_id
NEXT_PUBLIC_EMAILJS_PUBLIC_KEY=your_public_key
```

Execute o projeto:

```bash
npm run dev
```
